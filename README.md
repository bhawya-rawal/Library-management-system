# Library Management System

This is a simple Library Management System built with Java and Spring Boot.

## Local Setup

1.  **Prerequisites:**
    *   JDK 17
    *   Maven
    *   An IDE like VS Code.

2.  **Clone the repository:**
    ```bash
    git clone https://github.com/bhawya-rawal/Library-management-system.git
    ```

3.  **Build the project:**
    ```bash
    mvn clean install
    ```

4.  **Run the application:**
    ```bash
    mvn spring-boot:run
    ```

5.  **Access the application:**
    Open your browser and navigate to `http://localhost:9080`

6.  **Admin Credentials:**
    *   **Username:** `admin@admin.in`
    *   **Password:** `Temp123`

## Features

*   **Book Management:** Add, update, view, and delete books.
*   **Author Management:** Add, update, view, and delete authors.
*   **Category Management:** Add, update, view, and delete categories.
*   **Publisher Management:** Add, update, view, and delete publishers.
*   **Data Export:** Export library data to CSV and PDF formats.
